Organisation du cours

Déroulement d'une semaine type

Chaque semaine, vous aurez du travail préparatoire pour découvrir la matière par vous-même. Il s'agira d'une page web à lire, d'une video à suivre et/ou d'exercices à faire. Cela représente environ 2 heures de travail.

Ensuite pendant la séance, vous pourrez :

  • compléter et valider vos exercices avec le professeur
  • poser des questions
  • compléter les travaux évalués
L'ensemble du travail demandé doit être fait pendant la période.

Installation à la maison

À la maison, vous devez installer Java Dev Kit et l'environnement de développement Intellij IDEA

Votre propre application mobile

Vous devez développer votre propre application mobile au cours de la session. Les 3 phases du projet porteront sur cette application.

  1. Plusieurs activités, librairies de support
  2. Mise en page et besoins de service
  3. Navigation
  4. Réseau et asynchronisme
  5. Intégration / GUI attente
  6. Intégration / révision
  7. Intra
  8. Sérialisation de données HTTP
  9. Serveur et web service en Java (nouvelle version Jersey)
  10. Mot de passe, token, cookie et sécurité
  11. Réseau et asynchronisme (bis) Retrofit et autres
  12. Intégration
  13. Portrait paysage
  14. Révisions
  15. Final

Semaine 13 : interface adaptive, mode portrait et paysage

Évaluations

Avant la séance (2h)

Regardez l'exemple et en enlevant des morceaux de code, identifiez quels sont les éléments essentiels au bon fonctionnement.

Essayez de mettre en oeuvre les étapes suivantes

  • Layout Créez 2 layouts avec le même nom, un par défaut et un pour le mode paysage. Ajouter un TextView avec une couleur de fond dans celui pour paysage. Testez en tournant la tablette pour voir si le layout change.
  • Layout Ajouter le éléments nécessaire pour remplacer le TextView dans le layout paysage
  • Code Calculer une variable indiquant le mode dans le onCreate de l'Activity. Afficher le dans un Toast ou un Log et testez.
  • Code Insérez un if dans chaque morceau de code réagissant à la partie commune. Vous devrez effectuer l'action appropriée pour
    • mode portrait : partir une autre activité en lui passant ce qu'il faut
    • mode paysage : mettre à jour l'information dans la partie exclusive paysage
Pendant la séance en classe

Démonstration de création étape par étape d'une interface adaptive.

Pendant la séance de laboratoire
